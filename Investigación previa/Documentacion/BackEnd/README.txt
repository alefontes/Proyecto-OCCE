Para abrirlo hay que arrancar el main OcceAplication.java y ya que este funcionando.
Ingresan esta URL para entrar a Swagger UI y poder verlo: 

http://localhost:8080/swagger-ui/index.html

http://localhost:8080/v3/api-docs

y para descargar un archivo yaml es:
http://localhost:8080/v3/api-docs.yaml
